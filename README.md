# Chrome Extension: Chrome Helper Tool

## Giới thiệu
Đây là một Chrome Extension hỗ trợ người dùng trên trình duyệt. Extension này bao gồm chức năng chính:

1. **BookMark Custom**  
   - Đồng bộ với Bookmark của web
   - Cho phép điều chỉnh với UI đơn giản hơn

2. **Tab Group Manager**  
   - Quản lý các nhóm tab đang mở.  
   - Dễ dàng tạo, đổi tên, xóa hoặc di chuyển tab giữa các nhóm.  
   - Giúp duy trì không gian làm việc gọn gàng và khoa học.  

3. **Sidebar Note**  
   - Cho phép note thông tin ngay trên trình duyệt

4. **Sticky Note**
   - Cho phép dán note lên các trang web khác nhau (một vài trang bị hạn chế)

5. **Water Remider**
   - Nhắc nhở uống nước.
   - Tracking lượng nước đã uống trong ngày.

6. **Screen Tool**
   - Recording Screen
   - Capture Screen
   - Download image + video
   - Simple snipping Tool

7. **Eye reminder**
   - Simple eye relax reminder

## Cài đặt
1. Tải source code về máy.  
2. Mở Chrome → `chrome://extensions/` → Bật **Developer mode**.  
3. Nhấn **Load unpacked** và chọn thư mục chứa extension.  

## Lưu ý
- Extension này hoạt động tốt trên các phiên bản Chrome hiện đại.  
- Một số tính năng cần cấp quyền truy cập bookmark và tab.
