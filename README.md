# 🚀 Browser Boost - Comprehensive Browser Assistant

![logo](https://raw.githubusercontent.com/notepower2k1/ChromeHelperTool/refs/heads/main/icon128.png)

**Browser Boost** is a powerful browser extension designed to optimize your web experience, boost productivity, and prioritize your digital well-being. Optimized with a **Modern HSL Design System**, it provides a sleek, intuitive, and high-performance interface.

---

## ✨ Key Features

### 1. 📂 Advanced Tab Manager
*   **Smart Organization**: View and manage all open tabs in a clean list or colored groups.
*   **Quick Actions**: Create groups, rename, change colors, or close multiple tabs with a single click.
*   **Optimization**: Keep your workspace tidy and efficient.

### 2. 📝 Hybrid Note System
*   **Premium Sidebar Note**: A Rich Text editor integrated directly into the sidebar. Supports formatting, text colors, links, and smart search.
*   **Sticky Notes**: Attach mini-notes to any webpage. Supports dragging, resizing, and smooth entry/exit animations.
*   **Quick Extract**: Seamlessly transfer content from Sidebar Notes to Sticky Notes for better focus.

### 3. 🎨 Expert Color Picker (New!)
*   **On-Page EyeDropper**: Pick any color directly from the website you are browsing using the EyeDropper API.
*   **Floating Widget**: A compact, draggable UI that stays out of your way.
*   **Multi-Format Support**: Instantly get HEX, RGB, HSL, and HSV values.
*   **Gradient Preview**: Generates elegant linear gradients based on your selected color.

### 4. ⌨️ Quick Commands
*   **Custom Scripts**: Save and manage your frequently used code snippets or text commands.
*   **Formatted View**: View commands in a professional code-block style with multiline support.
*   **One-Click Copy**: Instantly copy any command to your clipboard.

### 5. 📋 Clipboard History Manager
*   **Smart Tracking**: Keeps an encrypted history of your copied text and **images**.
*   **Image Previews**: View copied images directly in the history list.
*   **Auto-Save Control**: Toggle automatic clipboard tracking on/off via the new UI controls.

### 6. 🎥 Pro Screen Tools
*   **Screen Recorder**: High-quality screen video recording.
*   **YouTube-style Preview**: Professional video preview with Play/Pause, 10s skip, and progress tracking.
*   **Advanced Capture Area**: Capture and edit screenshots instantly with:
    *   Drawing tools & shapes (Rectangle, Circle).
    *   Text annotations (Inter font).
    *   **Blur Tool**: Easily mask sensitive information.

### 7. 🌿 Digital Health Reminders
*   **Water Reminder**: Periodic hydration alerts with an elegant "floating drop" interface. Track your daily intake progress.
*   **Eye Relax**: Protect your vision with the 20-20-20 rule. Features a modern progress ring to guide your breaks.

## 🛠 Installation
1.  Download the **Source Code**.
2.  Open Chrome and navigate to: `chrome://extensions/`.
3.  Enable **Developer mode** (top right corner).
4.  Click **Load unpacked** and select the extension's root directory.

---

## 🔒 Permissions
This extension requires the following permissions to function optimally:
- `tabs` & `tabGroups`: For tab management features.
- `bookmarks`: For cross-device bookmark syncing.
- `storage`: To securely store your notes and preferences.
- `notifications`: For health and task alerts.
- `contextMenus`: For quick "Save to Command" right-click actions.
- `unlimitedStorage`: To accommodate large clipboard history (including images).
